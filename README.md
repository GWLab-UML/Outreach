# GW Lab Outreach
Repository for collection of science/marine focused outreach activities for K-12 age groups.

- Each folder is an individual activity with associated worksheets/materials
- Use the [Activity Template](https://github.com/GWLab-UML/Outreach/blob/main/ActivityTemplate.md) for adding new activities!

- If you would like to edit an activity, please submit an [issue](https://github.com/GWLab-UML/Outreach/issues) to request changes or submit the changes yourself (make a new branch, make your changes, and request @brookewicz in your pull request)
